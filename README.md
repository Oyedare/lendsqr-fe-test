## Run Locally

Clone the project

```bash
  git clone https://github.com/Oyedare/lendsqr-fe-test.git
```

Go to the project directory

```bash
  cd lendsqr-fe-test
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Start the build

```bash
  npm run build
```


## Tech Stack

**Client:** NextJS, TypeScript,Sass

**Api/Database:** Mock Api
